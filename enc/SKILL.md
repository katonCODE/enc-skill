# Easy and Concise

Use this skill to summarize the current chat or session into a short, clear digest.

## Goal

Create a digest that is easy to understand for someone who did not read the full chat.

## What to include

Include only visible, user-relevant information:

- What was changed
- What was created
- What was decided
- Important commands, files, or outputs
- Open questions or next steps

## What to avoid

Do not include:

- Private reasoning
- Hidden chain-of-thought
- Unimportant back-and-forth
- Overly detailed explanations
- Speculation

## Output format

Use this format:

### Summary

One or two short sentences.

### Changes / Work Done

- ...

### Decisions

- ...

### Open Questions / Next Steps

- ...

## Style

Be clear, concise, and direct.

Use plain language.

Prefer short bullets.

If nothing important changed, say that clearly.
